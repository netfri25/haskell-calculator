# haskell-calculator
simple calculator with operator precedence and brackets support implemented in haskell

### quickstart
make sure you have ```ghc``` and ```rlwrap``` installed (they can be cound in apt).

#### compiling
```sh
chmod +x compile.sh
./compile.sh
```

#### running
```sh
rlwrap ./calculator
```

#### more info
type ```exit``` or ```quit``` to terminate the program.
